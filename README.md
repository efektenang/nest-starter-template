## Description

[Nest.js](https://docs.nestjs.com) framework TypeScript Documentation.

## Installation

Clone the project

```bash
$ git clone https://github.com/efektenang/nest-starter-template.git
```

Go to the project directory

```bash
$ cd my-project
```

Install dependencies


```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
